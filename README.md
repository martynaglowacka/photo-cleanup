# PhotoCleanup

Intelligent photo library organizer: clusters near-duplicate photos, ranks the
photos within each cluster, and recommends which to keep. Nothing is ever
auto-deleted — "delete" candidates are moved to a `delete_candidates/` folder
for you to review and trash yourself.

**Status:** Weekend 1 — clustering pipeline (in progress).

## Setup

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```
